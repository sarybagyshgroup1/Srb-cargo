📋 Группы СИР — как выбрать

Группа 2 (рекомендуется) — развивающиеся страны в рамках GSP. Применяется для большинства товаров из Китая, Турции, Индии, Египта, Азии. Это ОСНОВНАЯ группа для карго-импорта через КР.

Группа 1 — без преференций / развитые страны (ЕС, США, Япония, Южная Корея). Самая высокая ставка СИР.

Группа 3 — наименее развитые страны (Афганистан, некоторые страны Африки). Самая низкая ставка.

🧮 Формулы расчёта
Минимальный инвойс
= Вес × СИР (выбранная группа)
Таможенная база
= MAX(Инвойс, Вес × СИР)
Пошлина (ВП)
= База × ВП%
НДС
= (База + Пошлина) × 12%
Таможенный сбор
= База × 0.4%
КАССА
= Пошлина + НДС + Сбор
