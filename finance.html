Структура Google Sheets

Таблица должна называться «Финансы» и содержать следующие листы:

СчетаA: Название, B: Тип (личный/бизнес), C: Сумма
КредитыA: Название, B: Банк, C: Изначально, D: Выплачено, E: Остаток, F: Ежем.платёж
ОбязательстваA: Название, B: Сумма, C: Счёт списания, D: Дата
ЛичныеA: Дата (ДД.ММ.ГГГГ), B: Описание, C: Категория, D: Приход, E: Расход, F: Счёт
БизнесA: Дата, B: Описание, C: Категория, D: Приход, E: Расход, F: Счёт
ЦелиA: Название, B: Период, C: Целевая сумма, D: Текущая, E: Срок, F: Цвет (hex)
📋 Скопировать шаблон Google Sheets
* После копирования: Файл → Поделиться → Доступ по ссылке → «Все, у кого есть ссылка» — просмотр
Инструкция — 3 шага
1
Скопируй шаблон Google Sheets выше. Заполни листы своими данными — счета, кредиты, транзакции.
2
Открой доступ: Файл → Поделиться → Доступ по ссылке → Просмотр. Скопируй ID из URL (длинный код между /d/ и /edit).
3
Вставь ID в поле выше → Сохранить. Сайт будет автоматически подтягивать данные при каждом открытии.
